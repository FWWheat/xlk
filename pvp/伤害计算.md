# 洛克王国世界 · PVP 伤害计算总结

> **游戏区分**：本文档只针对《**洛克王国：世界**》（腾讯 / 魔方工作室，2026 年 3 月 26 日公测的新手游），与早期的《洛克王国》（4399 / 腾讯页游 / 端游）**不是同一款游戏**。两者虽共享 IP，但战斗公式、属性体系、天气机制均有显著差异——本文不引用老洛克王国的 Fandom/灰机 Wiki 数据（例如 21 系含神属性、星云/血月/陨石天气、酷猫 1,377,344 示例等），这些都属于老洛克王国，不适用于《世界》。

---

## 一、整体定位：为什么像宝可梦

《洛克王国：世界》的核心框架与宝可梦几乎无缝衔接：

- **六维面板**：精力（HP）、物攻、魔攻、物防、魔抗、速度。
- **养成三件套**：种族值 + 个体值 + 努力值 + 性格 + 等级。
- **18 系属性**（常规属性体系，非老洛克的 21 系）。
- PVP **拉至统一等级**，胜负由配置、技能、战术决定。

因此《世界》的伤害计算逻辑与宝可梦高度相似，但公测期官方未公布纸面公式，目前公开的完整文字公式主要散见于 B 站 UP 主的实测视频中（如"府若"的伤害计算细节与加点收益详解）。以下内容综合社区实测与攻略整理。

---

## 二、六维面板计算公式

### 精力（HP）
```
HP = (种族值 × 2 + 个体值 + 努力值 ÷ 4) × 等级 ÷ 100 + 等级 + 10
```

### 物攻 / 魔攻 / 物防 / 魔抗 / 速度
```
属性值 = [(种族值 × 2 + 个体值 + 努力值 ÷ 4) × 等级 ÷ 100 + 5] × 性格修正
```

| 组成 | 说明 |
| --- | --- |
| **种族值** | 先天固定，每只精灵物种唯一。例：迪莫 410，罗隐 619 |
| **个体值（天分）** | 分四档：一般般 / 还不错 / 相当好 / 了不起，不可后天更改 |
| **努力值** | 果实投喂获得，单项 ≤ 255，总和 ≤ 510（推荐输出型 252 攻 + 252 速） |
| **性格修正** | 25 种性格，非平衡型对两项面板分别 +10% / −10% |
| **血脉魔法** | 后天养成，可临时强化同系精灵 |

## 三、伤害主公式（社区整理版）

公测期官方未发布明文公式，但根据实测与 B 站攻略视频推断，骨架与宝可梦一致：

```
伤害 = ⌊(攻方等级 × 0.4 + 2) × 威力 × 攻击 ÷ 防御 ÷ 50 + 2⌋
       × 本系加成 × 属性克制 × 暴击 × 随机数
```

> 注：`(等级×0.4+2)` 与宝可梦公式中的 `(2×等级÷5+2)` 在数值上等价。

| 参数 | 说明 |
| --- | --- |
| **威力** | 技能基础威力；变化技/固定伤害技不走此公式 |
| **攻击** | 计算能力等级变化后的物攻或魔攻 |
| **防御** | 计算能力等级变化后的物防或魔抗 |
| **本系加成（STAB）** | 技能与宠物自身属性相同时通常 ×1.5（参考宝可梦体系，实机以游戏为准） |
| **属性克制** | 参见第四节倍率表 |
| **暴击** | 触发时通常 ×1.5（具体倍率社区尚有争议，部分技能专属暴击倍率不同） |
| **随机数** | 单次攻击的伤害浮动区间（宝可梦一般为 0.85~1.00，《世界》实测接近） |

> **重要提醒**：上述公式与倍率系社区实测推断，官方未公开完整纸面公式。做精确计算前建议交叉对比 B 站实测视频，或通过游戏内实战验证。

## 四、属性克制倍率（18 系）

《世界》使用 18 种常规属性，单属性之间遵循克制 / 抵抗 / 免疫关系。

### 对单属性精灵

| 情况 | 倍率 |
| --- | --- |
| 克制 | ×2 |
| 正常 | ×1 |
| 抵抗 | ×1/2 |
| 免疫 | ×0 |

### 对双属性精灵（《世界》独有的 ×1.5 档）

| 情况 | 倍率 |
| --- | --- |
| 两个属性都被克制 | **×4** |
| 一个属性被克制 + 另一属性正常 | **×2** |
| 一个属性被克制 + 另一属性抵抗 | **×1.5** ← 手游新增档位 |
| 两个属性都正常 | ×1 |
| 一个属性抵抗 + 另一属性正常 | ×1/2 |
| 两个属性都抵抗 | ×1/4 |
| 任一属性免疫 | ×0 |

> 与老洛克王国的"×3 / ×1/3（双重克制/抵抗）"机制不同，《世界》按照宝可梦的**倍率乘积**方式结算，并额外加入了一克一抗的 ×1.5 档。

### 实例

- 草系技能打"水 + 地"双属性精灵 → ×4。
- 岩石 + 地面精灵被草/水技能打 → ×4。

## 五、PVP 等级与机制特性

### 5.1 统一等级
- PVP 入场时所有精灵**强制拉至同一等级**，种族值照常生效，但输赢主要由**个体值、性格、努力值、技能组、队伍构筑**决定。

### 5.2 听话等级（PVE 相关）
- 精灵有"听话等级"软上限，超过玩家徽章控制范围的精灵可能**不听指挥**。野外捕捉高级精灵仍有一定概率听话，允许早期越级攻略。

### 5.3 印记 vs 减益
| 机制 | 影响范围 | 切换精灵是否保留 |
| --- | --- | --- |
| **印记** | 全队（类似 buff） | **保留**；最多同存 1 正 1 负 |
| **减益** | 单只精灵 | 切换即清除 |

这是《世界》PVP 极具特色的设计，直接影响开局博弈与切换节奏。

### 5.4 先手技能
- 梦魇来临：先手 +5
- 铁骑觉醒：先手 +7
- 反制 / 固守类技能可反弹先手、抵御爆发伤害

### 5.5 角色定位
PVP 常规 3~6 位阵容中，精灵按功能划分为：
- **主 C**（1~2 只核心输出）
- **辅助 / 增益手**（印记、加状态）
- **减益手 / 反印**（移除对方状态）

## 六、天气与场地（《世界》实装部分）

《世界》目前主要的天气为通过技能召唤的场控类，**并未**像老洛克那样拥有"星云 / 血月 / 陨石 / 圣光"等扩展天气：

- **下雨**：水系技能威力提升，火系威力下降。
- **晴天 / 暴晒**：火系技能威力提升，水系威力下降。
- **沙暴**：非特定属性回合末受损；部分属性免疫。
- **暴风雪 / 冰雹**：冰系受益；非冰系回合末受损。

> 具体数值以游戏内实际显示为准。《世界》更强调**特性联动**而非"反转克制"这类极端场控。

---

## 七、特性（被动）对伤害的影响

《世界》的特性设计非常激进，是 PVP 胜负的重要变量：

- **天气召唤型**：如彩蝶鲨（首领形态）登场自动召【雨天】，联动水系阵容。
- **跨属性状态叠加**：例如火系输出附加冻结、冰系输出附加点燃等，形成"套娃"玩法（宝可梦没有的设计）。
- **技能复制型**：如帅帅魔偶可随机复制队友技能，做战术中枢。
- **传动 / 顺发型**：如帕帕斯卡让某一技能位顺发，英雄技能因传动属性持续变换。
- **毒系爆毒流**：毒锅锅的爆毒玩法在《世界》为主流体系之一。
- **文官 / 武官应对成功机制**：文官携带"一拳"类技能，成功应对后伤害 ×10；武官则靠多段连击放大应对成功倍率。

---

## 八、本系加成（STAB）与同系减伤

- 本系加成倍率通常按 **×1.5** 计（沿用宝可梦，实机需以游戏为准）。
- 同系技能对同系精灵存在**减伤判定**，因此选择**反属性兜底**往往更稳。
  - 例：罗隐是恶系，打恶系用自身土系技能反而不减伤。

---

## 九、暴击与随机

- 暴击触发时伤害放大（社区推测 ×1.5，部分资料为 ×2，以实机为准）。
- 每次伤害结算附带一次随机数波动（宝可梦惯例 0.85~1.00），这是"同配置不同次伤害不一致"的主因。

---

## 十、完整结算流程

1. 计算双方六维面板（种族值 + 个体值 + 努力值 + 性格 + 等级）。
2. 叠加场上强化 / 弱化（+1 ~ +6 / −1 ~ −6 等级）。
3. 命中判定（受命中率、闪避率、特性、属性免疫影响）。
4. 代入主公式：`⌊(等级·0.4+2)×威力×攻÷防÷50+2⌋`。
5. 乘 **本系加成**（若技能属性与精灵属性同）。
6. 乘 **属性克制**（参照第四节倍率表）。
7. 乘 **天气 / 场地 / 特性** 加成。
8. 乘 **暴击** 加成。
9. 乘 **随机数** 浮动。
10. 最终向下取整，`<1` 修正为 `1`。

## 十一、与老《洛克王国》的主要区别（避免混淆）

| 项目 | 洛克王国世界（新手游） | 洛克王国（老页游/端游） |
| --- | --- | --- |
| 属性数量 | **18 系** | 21 系（含神草 / 神水 / 神火） |
| 天气 | 下雨 / 晴天 / 沙暴 / 冰雹 等常规 | 还包含星云、血月、陨石、圣光等特殊天气 |
| 双属性克制 | ×4 / ×2 / **×1.5** / ×1 / ×0.5 / ×0.25 | ×4 / ×2 / ×3 (双重克制) / ×1/3 (双重抵抗) |
| 数值体系 | 种族值 + 个体值 + 努力值 + 性格 | 种族值 + 个体值 + 学习力 + 性格（近似但细节不同） |
| PVP 等级 | 拉至统一等级 | 各等级段分别匹配 |
| 特色机制 | **印记 / 减益 / 听话等级 / 跨属性状态叠加** | 英雄魂印、神系克制等 |
| 权威数据源 | TapTap 官方论坛、GameKee Wiki、B 站"府若"等 UP 主 | Fandom Wiki、灰机 Wiki、4399 攻略 |
